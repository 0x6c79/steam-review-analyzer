# 仪表板完整指南

详细的 Streamlit 仪表板功能和操作指南。

## 快速启动

```bash
streamlit run src/steam_review/dashboard/dashboard.py
```

访问 http://localhost:8501

## 主要功能

### 📊 概览页面 (Overview)

- 总体统计信息和关键指标
- 评分分布图表
- 情感趋势可视化
- 快速数据汇总

### 🔧 爬虫页面 (Scraper)

- 输入 Steam 游戏 App ID
- 爬取新评论
- 增量更新已有数据
- 进度显示和日志

### 📈 分析页面 (Analysis)

- 情感分析结果
- 关键词频率统计
- 词云可视化
- 主题建模结果
- 时间序列分析

### 🔬 高级分析 (Advanced Analytics)

- 玩家分割分析
- 版本影响分析
- 游戏对比
- 质量评分排名
- 推荐评论

### 💾 数据库 (Database)

- 浏览所有评论
- 搜索和过滤
- 数据导出（CSV/Excel/JSON/PDF）

### ⚙️ 设置 (Settings)

- 缓存管理
- 显示选项
- 性能设置

## 性能优化提示

1. **首次启动**: 首次访问需要 30-60 秒进行分析
2. **缓存利用**: 后续访问从缓存读取，<1 秒加载
3. **手动刷新**: 如果需要重新分析，点击"清除缓存"按钮
4. **数据量**: 建议 100-100,000 条评论以获得最佳性能
